Favicon Information
==================

The Give Protocol logo is used as the favicon for the documentation site.

Current Setup:
- logo.svg - The main SVG logo (works as favicon in modern browsers)
- favicon.ico - Placeholder file for legacy browser support

To generate PNG favicons from the SVG:
1. favicon-32x32.png - 32x32 pixels
2. favicon-16x16.png - 16x16 pixels  
3. apple-touch-icon.png - 180x180 pixels

The SVG favicon will work in most modern browsers. The PNG versions are
referenced in the HTML but not required for the favicon to display.
